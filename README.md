# i-keep-sync

## Project Status
Scraping & saving to csv works.
Enhancements are on hold but listed in Roadmap section.

## Description
This project is used for syncing the products with certifications on iKeepSafe website with the certification badges in the LP library.

It scrapes data from the iKeepSafe website and saves data in a csv file named with today's date.


## Installation
See environment.yml file for dependancies. 

## Usage
1. In terminal, run 'python main.py'.
2. CSV of scraped data is in data/ directory.
